Aplicación WEB realizado por Nahuel Miralpeix Escalante para el ciclo de Desarrollo de Aplicaciones Multiplataforma

Aplicación realizada para el módulo de DI, hecha en ANGULAR que consta de:

  • Componente home: representa la entrega inicial de la aplicación
  • Componente Ciclos: representa tantas cartas como ciclos existan en el servicio CiclosService
  • Componente Asignaturas: representa tantas cartas como asignaturas haya en el servicio AsignaturasService. Adicionalmente se cuenta con un filtro para poder ver asignaturas especifícas
  • Componente Asignatura: representa el aspecto que tendrá una carta de las de arriba mencionada
  • Coponente DetalleCiclo: representa el detalle de un ciclo, donde se mostrarán los conocimientos necesarios para poder aprobar el ciclo y todas las asignaturas que lo componen
  • Componente NavBar: representa la barra de navegación superior, donde se muestra un menú con las siguientes posibilidades
    • Inicio: llevará al componente Home
    • Ciclos: llevará al componente Ciclos
    • Asignaturas: llevará al componente Asignaturas
  • Servicios: Se deberá crear al menos dos servicios de datos tanto para los ciclos como para las asignaturas
  • Pipes: se deberá crear al menos dos directivas personalizadas que realice alguna transformación en la aplicación
